# BackEndUNAP

## Documentation
https://documenter.getpostman.com/view/23245717/2s8ZDSajer

## Development

```sh
npm install
```

Start the development asset server and the Express server by running:

```sh
npm run dev
```

initialize with seed with next command:

```sh
npm run seed
```

## Deployment

Then run the app in production mode:

```sh
npm run start
```
